# Multiply Calculator

A small interactive web application where the user can enter two numbers and calculate their product.

## Features

- Multiplies two user-provided numbers
- Uses parameters and global state variables
- Reads input values via `this.value`
- Re-renders the view using a simple MVC-inspired structure

## Focus

- Parameters
- Variables (state management)
- `this.value`
- Functions with return values
- DOM updates via re-rendering

## Run

Open `index.html` with Live Server.
